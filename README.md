BackupShell is just that. A shell script for creating backups of a system using tar. 

## Features
1. Retention policy
2. Compressed backup
3. Exclude list

## Install
1. Copy to /usr/local/lib64/ 
2. Configure script options in backup.py
3. Schedule a cron job to run backup.py
